# webpack5-demo



webpack5学习demo

## 项目启动和打包

```bash
# 安装依赖包
npm install

# 启动开发环境
npm run dev

# 项目打包
npm run build
```

## 支持特性

<!--

- ✅ 使用axios请求数据
- ✅ 通过nodejs获取项目git信息
- ✅ 使用eslint + prettier实现代码风格校验统一以及保存自动格式化
- ✅ 使用husky规范commit提交记录
- ✅ nginx容器化部署，实现打包一套代码，应用多套环境，比如，生产环境、UAT环境、测试环境、开发环境等
- ❌ 集成typescript
- ❌ 集成jest实现单元测试
- ✅ 集成cypress实现自动化测试
- ❌ 自定义埋点
- ❌ 前端异常日志收集捕获
